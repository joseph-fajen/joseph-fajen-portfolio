## Marlowe Runtime API — Input Output Global (2022–2025)

The Marlowe Runtime is a 7-service architecture for deploying and managing smart contracts on Cardano. I designed and wrote the full API reference documentation alongside the engineering team.

**What I built:**
- 91-schema OpenAPI specification covering the complete Marlowe Runtime service layer
- Integration with Docusaurus for rendering and navigation
- Weekly engineering syncs to validate accuracy across 7 microservices

**Live:** [docs.marlowe.iohk.io](https://docs.marlowe.iohk.io/docs/introduction)

## AJA Video REST API — AJA Video Systems (2016–2022)

AJA Video had no developer documentation when I joined. I initiated and owned the company's first REST API docs.

**What I built:**
- 8-chapter developer guide covering 7 products across the Ki Pro and HELO lines
- Partnered directly with engineers through iterative review cycles
- See: https://gitlab.aja.com/pub/rest_api
- Information architecture designed for both broadcast engineers and developers

**Scope:** First-ever developer documentation at the company.
