## The Project

[Blockfrost](https://blockfrost.io/) is a blockchain API provider for Cardano. During my time at Input Output Global, for a brief period (two to three months), I was assigned to assess and improve their documentation. The team had not had a technical writer, so the documentation up to that point had been a side project of the developer team. 

---

## What I Delivered

### 1. Information Architecture Improvements

The Blockfrost platform documentation needed more general descriptions and definitions of concepts and offerings. It also needed restructuring to better serve developers at different stages of adoption. 

My workflow included these steps:

- Analyze the existing documentation structure against user journeys
- Verify documentation accuracy against the actual platform behavior
- Clarify developer-facing docs to help early adopters get up and running faster
- Improve navigation and discoverability

**Result:** [platform.blockfrost.io](https://platform.blockfrost.io/)

### 2. AI-Powered FAQ Development

The FAQ section required a different approach. I had inherited a FAQ page that was blank. But there were rich conversations going on between early adopters and the platform developers in the project's Discord channels. I wanted to find a way to capture the nuggets of all those exchanges and channel them into the FAQ page. 

So I built a tool to analyze these real user conversations, identify the patterns in the questions being asked, and create first drafts of questions and answers that represented the key insights from the Discord conversations. Using this approach accelerated what would otherwise have taken more time to do by hand. 

After careful reviewing and scrubbing of the tool outputs, we had a rich, categorized FAQ page. 

**Result:** [platform.blockfrost.io/faq](https://platform.blockfrost.io/faq)

> The tool I built, [ChatDoc-InsightMiner-PromptLab](https://github.com/joseph-fajen/ChatDoc-InsightMiner-PromptLab) toolkit, is open source on GitHub, and applicable to any documentation project where user conversation data is available.
